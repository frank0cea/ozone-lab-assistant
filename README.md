# 硫代硫酸钠标定助手

一个无需安装、可直接在手机或电脑浏览器中使用的单功能网页，用于按重铬酸钾法标定硫代硫酸钠滴定液。

## 当前功能

- 记录重铬酸钾标准液浓度、取用体积和溶液批次
- 6 min 避光稳定倒计时，支持暂停、继续和刷新后恢复
- 两组平行样的滴定管初始读数和终点读数
- 自动计算消耗体积与硫代硫酸钠实际浓度
- 自动计算两组平行样平均浓度及相对差，按 2% 判断
- 每组30 s终点不返蓝确认
- 本机历史记录、CSV导出、JSON备份与导入
- 自动保存未完成的当前记录草稿

## 计算公式

```text
V1 = Vfinal - Vinitial
B = 6 × N × V2 / V1
```

- `B`：硫代硫酸钠滴定液实际浓度，mol/L
- `N`：重铬酸钾标准溶液浓度，mol/L
- `V1`：硫代硫酸钠滴定液消耗体积，mL
- `V2`：重铬酸钾标准溶液取用体积，mL

平行结果相对差按极差除以平均值计算。相对差不超过2%时，程序将平均浓度作为推荐记录值。

## 使用

直接访问 [GitHub Pages 网页](https://frank0cea.github.io/ozone-lab-assistant/)，或下载仓库后双击 `index.html`。所有实验数据仅保存在当前设备、当前浏览器中；程序不连接数据库，也不上传实验数据。

## 验证

```bash
npm run check
```

程序用于减少换算和记录错误，不能代替实验室安全规程、标准方法原文或人工复核。
