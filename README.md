# OzoneLab 臭氧实验助手

一个无需安装、可直接在浏览器中打开的本地网页程序，用于“臭氧预处理 + Ca²⁺/Mg²⁺ + 超滤膜污染”实验中的重复计算、步骤计时和记录整理。

## 当前功能

- 通用稀释与摩尔储备液称量
- Ca²⁺ / Mg²⁺ 储备液投加体积
- 直接曝气所需臭氧质量与曝气时间
- 臭氧水投加体积、总体积占比及投加前 DOC 修正
- 两级 KI 尾气吸收、臭氧转移效率、`Dapp` 与 `Dtransfer`
- 瞬时膜通量 `J`、归一化通量 `J/J₀` 和通量恢复率 `FRR`
- 直接曝气、臭氧水两套实验步骤模板
- 步骤顺序检查、可调计时器、到时提示及完成时间记录
- 实验记录和计算/流程快照
- CSV 导出、JSON 备份与合并导入

## 使用方法

1. 下载或克隆整个项目文件夹。
2. 双击 `index.html`，用 Chrome、Edge 或其他现代浏览器打开。
3. 从“计算中心”完成参数换算，再进入“实验流程”。
4. 实验结束后在“实验记录”中保存，并定期导出 JSON 备份。

程序数据保存在当前浏览器的本地存储中，不会自动上传。请不要随意移动项目文件夹或清理浏览器站点数据；重要实验完成后及时导出 JSON。

如浏览器限制直接打开本地文件，也可在项目目录运行：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 主要计算约定

### 直接曝气

```text
m_target = D_target × C_DOC × V_water
t = m_target / (P_O3 × η_transfer)
```

其中剂量单位为 `mg O₃/mg DOC`，DOC 为 `mg/L`，水样体积为 `L`，发生器产量为 `mg/min`。

### 臭氧水投加

```text
V_ozone-water = m_target / (C_ozone-water × R_retention)
```

臭氧水计入最终总体积。程序同时计算投加前基础污染液应达到的 DOC，避免加入臭氧水后把最终 DOC 稀释到目标值以下。

### 两级 KI 尾气吸收

```text
m_tail = 24 × C_thio × [(V1 - V0) + (V2 - V0)]
m_transfer = P_O3 × t - m_tail
Dtransfer = m_transfer / (C_DOC × V_water)
```

滴定体积按 mL 输入，且每个 KI 瓶分别扣除一次空白体积 `V0`。

### 膜通量

```text
J = ΔV / (A × Δt)
J/J0 = J / J0
FRR = Jr / J0 × 100%
```

## 验证

计算模块使用 Node.js 内置断言测试，不需要安装第三方依赖：

```bash
npm run check
```

## 注意

本程序用于减少换算和记录错误，不能替代臭氧安全规程、仪器说明书、标准分析方法或人工复核。发生器产量、转移效率和臭氧水浓度必须使用与本次实验条件一致的实测值。
